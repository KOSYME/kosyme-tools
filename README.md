# KOSYME Portfolio Allocation Diagnostic

A one-screen Next.js diagnostic tool for evaluating what a portfolio is exposed to and how it may behave under different structural conditions.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Frontend-only logic
- No backend, database, or external APIs

## Run Locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```
